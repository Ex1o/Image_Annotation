┌─────────────────────────────────────────────────────────────────┐
│ 🔐 AUTH FLOW                                                    │
└─────────────────────────────────────────────────────────────────┘

Registration:
  Form → /auth/register → Success → Redirect to Login

Login:
  Form → /auth/login → Get Tokens → Store in localStorage
  → Redirect to Home → Show User Menu

Protected Route Access:
  User navigates → ProtectedRoute checks tokens 
  → If valid: Allow → If not: Redirect to /login

Token Refresh:
  API call fails (401) → Axios interceptor catches
  → POST /auth/refresh → Get new tokens → Retry request

Logout:
  Click logout → Clear localStorage → Redirect to /login
